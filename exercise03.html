Four other psuedo classes & elements:

Psuedo Classes

  1. Simple Tooltip Hover. Hover over a div element to show a paragraph element (like a tooltip)
  2. CSS - The :first-child Pseudo-class. The :first-child pseudo-class matches a specified element that is the first child of another element.
  3. Match the first paragraph element. The selector matches any paragraph element that is the first child of any element
  4. CSS - The :lang Pseudo-class. The :lang pseudo-class allows you to define special rules for different languages.

Psuedo Elements

  1. CSS - The ::before Pseudo-element. The ::before pseudo-element can be used to insert some content before the content of an element.
  2. CSS - The ::after Pseudo-element. The ::after pseudo-element can be used to insert some content after the content of an element.
  3. CSS - The ::marker Pseudo-element. The ::marker pseudo-element selects the markers of list items.
  4. CSS - The ::selection Pseudo-element. he ::selection pseudo-element matches the portion of an element that is selected by a user.
